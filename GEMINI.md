# Gemini Context

To run all unit tests, use the `make test` command. Do not run the tests in the `./test` directory directly, as they are end-to-end tests and require a specific environment setup.

To fix linter errors, run `make lint` or `golangci-lint run`.

Use `make fumpt` to fix linter errors related to formatting.

Generally you can do a `make help` to see all available commands for helping you with development.
